## 3. Flow Designer 节点内容丢失、连线文字为空

- **所属模块**: Flow Designer / playground
- **问题描述**: Flow Designer 中节点显示丢了内容，连线上的文字也是空的，怀疑是取错了变量名
- **发现方式**: 人工测试
- **状态**: 已修复（三个层次的修复）
- **修复详情**:
  1. **Bindings 展开**: `designer-xyflow-edge.tsx` 和 `designer-xyflow-node.tsx` 中 edge/node renderData 未将嵌套数据（`props.data`、`config`）展开到顶层 scope
  2. **Root cause — 表达式预求值**: `designer-page.tsx` 的 `config` prop（包含 nodeType/edgeType 模板 schema）被 `expressionCompiler.compileValue()` 在页面级别递归求值。`${label}` 等模板表达式在页面 scope 中不存在对应变量，求值结果为 `undefined`，导致所有 nodeType body 变成 `{ type: "text", body: undefined }`，节点内容为空
  3. **Fix**: `DesignerPageRenderer` 改为从 `meta.templateNode.schema`（原始未编译 schema）读取 `config`，而非从 `props.props['config']`（已被表达式求值污染的 props）
- **关键文件**: `packages/flow-designer-renderers/src/designer-page.tsx:121-123`
- **验证**: 6/6 e2e 测试通过，84/84 flow-designer-renderers 单元测试通过，281/281 flux-compiler 单元测试通过
