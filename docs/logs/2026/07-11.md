# Development Log - 2026-07-11

### 2026-07-11 (feature: ERP 集成场景驱动的 Schema 缺口补齐 — 执行 docs/plans/2026-07-11-schema-gap-from-erp-integration-plan.md)

**背景**：nop-app-erp（337 实体）计划从 AMIS 迁移到 Flux。在对 view.xml 全量属性与 Flux 组件 schema 的逐项对照中识别出一组 Flux 尚未覆盖的 UI 行为。本日志记录该 plan 的 6 个 Phase 的完整执行。

**改动（按 Phase）**：

- **Phase 1 — Table `autoFillHeight`**
  - `packages/flux-renderers-data/src/schemas.ts` — `TableSchema` 新增 `autoFillHeight?: boolean | { height?: number; maxHeight?: number }`
  - `packages/flux-renderers-data/src/data-renderer-definitions.ts` — table fields 新增 `{ key: 'autoFillHeight', kind: 'prop' }`
  - `packages/flux-renderers-data/src/table-renderer/use-auto-fill-height.ts`（新增）— ResizeObserver 监听父容器，计算视口剩余高度；`{ height: N }` 固定高度、`{ maxHeight: N }` 最大高度、`true` 自动计算；父容器 clientHeight === 0 时 rAF 重试（最多 10 次）；loading true→false 重新测量；render-time 派生 fixed/disabled 情况，measured 高度经 ResizeObserver 异步回调更新（避免 react-hooks/set-state-in-effect）
  - `packages/flux-renderers-data/src/table-renderer.tsx` — 接入 hook，容器应用 heightStyle + `data-auto-fill-height` marker；与 `affixHeader` 共存（容器内 sticky 表头而非禁用，改进 amis 互斥行为）
  - `__tests__/table-auto-fill-height.test.tsx`（新增，6 tests）

- **Phase 2 — CRUD/Table `selection.toggleOnRowClick`**
  - `crud-schema.ts` `CrudSelectionConfig` + `schemas.ts` `TableSchema.rowSelection` 新增 `toggleOnRowClick?: boolean`；CRUD renderer 透传到底层 table
  - `table-body-row-rendering.tsx` — 行点击事件链：检查 `isClickOnInput`（input/textarea/button/a/checkbox/switch/role）→ 检查 `maxSelectionLength` 已满 → 执行自定义 `onRowClick` → 仅在实际 toggle 时 `preventDefault`（改进 amis 始终 preventDefault 阻止文本选择）；行 `<tr>` 新增 `data-row-toggleable` marker
  - `crud-renderer.tsx` — `selection.toggleOnRowClick` 透传到 `base.rowSelection`
  - `__tests__/toggle-on-row-click.test.tsx`（新增，8 tests）+ `crud-selection-drift-deletion.test.ts` 更新字段计数（4→6）

- **Phase 3 — Button `tooltipPlacement` + `countDown` / `countDownTpl` + `href`/`target`**
  - `schemas.ts` `ButtonSchema` 新增 `tooltipPlacement?: { side?, align? }`、`countDown?: number`、`countDownTpl?: string`、`href?: string`、`target?: string`
  - `button.tsx` — tooltipPlacement 透传到 `@nop-chaos/ui` Tooltip `side`/`align`；countDown 经 `setInterval` 驱动（基于 `Date.now()` 差值避免漂移），action 成功后（`await onClick` resolve）启动，倒计时期间 `disabled = true` + label 替换为模板渲染；localStorage 持久化 key = `flux-countdown-${pathname}-${id||name}`（仅显式 id/name 时持久化，改进 amis uuid 垃圾）；`href` 配置时渲染为 `<a>` 而非 `<button>`；`countDownTpl` 使用 `{timeLeft}` token（非 `${timeLeft}`，避免 Flux 表达式编译器拦截）
  - `basic-renderer-definitions.ts` — button fields 新增 tooltipPlacement/countDown/countDownTpl/href/target
  - `__tests__/button-tooltip-placement.test.tsx`（新增，4 tests）、`button-count-down.test.tsx`（新增，7 tests）、`button-href.test.tsx`（新增，2 tests）
  - **countDown 决策反转**：`button/design.md` 决策表从"不采纳"更新为"实现"，附推翻理由

- **Phase 4 — Page `asideResizable` / `asideMinWidth` / `asideMaxWidth` / `asideSticky`**
  - `schemas.ts` `PageSchema` 新增四个属性
  - `page.tsx` — `asideResizable: true` 时渲染 drag handle `data-slot="page-aside-resize-handle"`，pointer events + `setPointerCapture`（原生触摸+笔支持），宽度存 React local state，clamp 到 `[asideMinWidth ?? 200, asideMaxWidth ?? 600]`，`asidePosition: 'right'` dx 反转，右键忽略；`asideSticky: true` 应用 `position: sticky; top: 0; max-height: 100vh; overflow-y: auto`
  - `basic-renderer-definitions.ts` — page fields 新增四个属性
  - `__tests__/page-aside-resizable.test.tsx`（新增，7 tests）

- **Phase 5 — ApiSchema `responseType` / `downloadFileName` + blob-download 工具函数**
  - `packages/flux-core/src/types/schema-base-types.ts` — `ApiSchema` 新增 `responseType?: 'json'|'blob'|'text'`、`downloadFileName?: string`、`dataType?: 'json'|'form-data'|'form'`；`ExecutableApiRequest` 新增 `responseType`/`downloadFileName`
  - `packages/flux-runtime/src/async-data/request-runtime.ts` — `finalizeApiRequest`/`materializeApiRequest` 传播 responseType/downloadFileName 到 ExecutableApiRequest
  - `packages/flux-runtime/src/async-data/blob-download.ts`（新增）— `extractFilenameFromContentDisposition`（支持 RFC 5987 `filename*=UTF-8''`）、`resolveDownloadFilename`（downloadFileName 优先）、`downloadBlob`（object URL + `<a download>` + 40s 后 revoke，改进 100ms 竞态）、`normalizeBlobResponse`（JSON-in-blob 错误恢复 + 合成成功响应）；从 `packages/flux-runtime/src/index.ts` 导出
  - `__tests__/blob-download.test.ts`（新增，11 tests）
