# Manual Test Issues — 2026-05-11

## 1. tabs with form 切换后输入值丢失

- **所属模块**: playground / tabs lab / form
- **问题描述**: 在 `http://localhost:5173/#/lab/tabs` 的 `Tabs with forms` 场景中，先在一个 tab 内输入表单值，再切换到另一个 tab 后切回，之前输入的内容会丢失，看起来像表单草稿被重置了。
- **预期行为**: tabs 只切换可见面板，不应重置 tab 内 form 的本地值；切回原 tab 后应看到之前输入的内容仍然保留。
- **发现方式**: 人工测试
- **状态**: 已修复（`packages/ui/src/components/ui/tabs.tsx` 默认保持 inactive panel mounted；`packages/flux-renderers-basic/src/__tests__/basic-page-layout-structure.test.tsx` 补充 tabs 内 form 值保留回归测试）

## 4. Report Designer 面板无收起/展开按钮

- **所属模块**: playground / report-designer-demo
- **问题描述**: Report Designer playground 页面 (`/#/report-designer`) 的左侧字段面板和右侧 inspector 面板没有收起/展开按钮。左侧字段面板使用文本字符 `<`/`>` 作为切换按钮，右侧 inspector 完全没有收起能力。根本原因是 `ReportDesignerDemo` 使用手工 flex 三列布局而非共享的 `WorkbenchShell`。
- **预期行为**: 左右面板应有与 Flow Designer / Word Editor 一致的收起按钮（lucide chevron 图标，canvas 邻近边缘），收起后显示共享 collapsed rail。
- **发现方式**: 人工测试
- **状态**: 已修复（`apps/playground/src/pages/report-designer-demo.tsx` 重写为使用 `WorkbenchShell`；`tests/e2e/report-designer-demo.spec.ts` 更新为 `workbench-*` 选择器；e2e 7/7 通过）

## 6. Sheet tab 切换时抖动 + 样式不统一

- **所属模块**: spreadsheet-renderers / sheet-tab-bar
- **问题描述**: 多个 sheet tab 时点击切换会抖动，原因是 `.ss-sheet-tab[data-active]` 设置 `font-weight: 600` 而非活动 tab 为 400，字体粗细变化导致文本宽度变化从而布局偏移。同时 sheet tab 使用硬编码颜色（#fafafa 等渐变背景、Excel 风格圆角顶部），与应用其他部分的 design token 风格不统一。
- **预期行为**: 所有 tab 统一 `font-weight: 500`，切换无抖动；使用 CSS 变量（`--border`、`--nop-accent` 等）而非硬编码颜色；active 指示使用 `after::` 伪元素底部色条，与 UI 库 `Tabs variant="line"` 风格一致。
- **发现方式**: 人工测试
- **状态**: 已修复（`packages/spreadsheet-renderers/src/canvas-styles.css` 重写 sheet tab bar 样式）
