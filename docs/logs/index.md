# Development Log

Development log entries are organized by date, with one file per day.

## Structure

```
docs/logs/
├── 00-log-writing-guide.md
├── index.md
├── 2026/
│   ├── 03-31.md
│   ├── 03-30.md
│   └── ...
└── 2027/
    └── ...
```

## Usage

- Path convention: `docs/logs/{year}/{month}-{day}.md`
- Writing rules and entry format: `docs/logs/00-log-writing-guide.md`
