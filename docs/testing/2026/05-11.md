## 1. tabs with form 切换后输入值丢失

- **所属模块**: playground / tabs lab / form
- **问题描述**: 在 `http://localhost:5173/#/lab/tabs` 的 `Tabs with forms` 场景中，先在一个 tab 内输入表单值，再切换到另一个 tab 后切回，之前输入的内容会丢失，看起来像表单草稿被重置了。
- **预期行为**: tabs 只切换可见面板，不应重置 tab 内 form 的本地值；切回原 tab 后应看到之前输入的内容仍然保留。
- **发现方式**: 人工测试
- **状态**: 已修复（`packages/ui/src/components/ui/tabs.tsx` 默认保持 inactive panel mounted；`packages/flux-renderers-basic/src/__tests__/basic-page-layout-structure.test.tsx` 补充 tabs 内 form 值保留回归测试）

## 6. Sheet tab 切换时抖动 + 样式不统一

- **所属模块**: spreadsheet-renderers / sheet-tab-bar
- **问题描述**: 多个 sheet tab 时点击切换会抖动，原因是 `.ss-sheet-tab[data-active]` 设置 `font-weight: 600` 而非活动 tab 为 400，字体粗细变化导致文本宽度变化从而布局偏移。同时 sheet tab 使用硬编码颜色（#fafafa 等渐变背景、Excel 风格圆角顶部），与应用其他部分的 design token 风格不统一。
- **预期行为**: 所有 tab 统一 `font-weight: 500`，切换无抖动；使用 CSS 变量（`--border`、`--nop-accent` 等）而非硬编码颜色；active 指示使用 `after::` 伪元素底部色条，与 UI 库 `Tabs variant="line"` 风格一致。
- **发现方式**: 人工测试
- **状态**: 已修复（`packages/spreadsheet-renderers/src/canvas-styles.css` 重写 sheet tab bar 样式）

## 7. Report Designer 表格滚动区域不正确，header 和 sheet tab 被裁剪

- **所属模块**: playground / report-designer-demo
- **问题描述**: 在 `/#/report-designer` 页面中，表格的列头（A, B, C...）和底部 sheet tab 在滚动时被裁剪或不可见。根因是 `SpreadsheetGrid` 的 `.ss-grid-shell`（`flex: 1`）被包在一个 `display: block` 的 `overflow-hidden` div 中，flex 失效，grid 按内容撑到 722px 而非约束在 536px。
- **预期行为**: grid 应在可用空间内正确约束并滚动，header sticky 在顶部，sheet tab 固定在底部。
- **发现方式**: 人工测试
- **状态**: 已修复（移除中间 `overflow-hidden` wrapper，让 `.ss-grid-shell` 直接作为 `flex flex-col` 的子元素；同时给 `.ss-grid-shell` CSS 添加 `min-height: 0`）

## 8. Spreadsheet 功能全面审计（与 Excel 对比）

- **所属模块**: spreadsheet-renderers / spreadsheet-core
- **问题描述**: 人工对比 Excel 基本编辑功能，发现多项缺失。以下是完整审计结果。

### 已实现的功能

**键盘快捷键**：
| 快捷键 | 功能 |
|---|---|
| Arrow Up/Down/Left/Right | 单元格导航 |
| Enter | 进入编辑模式 / 保存编辑 |
| Escape | 取消编辑 |
| Ctrl+C/X/V | 复制/剪切/粘贴 |
| Ctrl+Z/Y | 撤销/重做 |
| Ctrl+B/I/U | 加粗/斜体/下划线 |
| Ctrl+F | 查找替换 |
| Delete/Backspace | 清除单元格 |
| 可打印字符 | 直接替换内容并进入编辑 |

**工具栏按钮（32个）**：

- 撤销/重做（2）
- 复制/剪切/粘贴/清除（4）
- 加粗/斜体/下划线（3）
- 左对齐/居中/右对齐（3）
- 背景色：黄/绿/蓝/无（4）
- 字体色：红/蓝/黑（3）
- 合并/取消合并/合并居中（3）
- 向下填充/向右填充（2）
- 插入行/删除行/插入列/删除列（4）
- 添加批注/查找替换（2）
- 冻结/取消冻结（2）

**鼠标交互**：

- 单击选中、双击编辑、拖拽选区
- 填充柄拖拽（仅向下/向右）、双击自动填充
- 列宽/行高拖拽调整
- 右键上下文菜单（复制/剪切/粘贴/清除/排序/筛选/合并/冻结/插入删除行列）
- 字段拖放（从字段面板拖到单元格）

| 16  | 单元格边框                     | CSS classes `.ss-border-solid/dashed/dotted/double` 存在但无工具栏按钮                                                                         |
| 17  | 单元格值同步为 fire-and-forget | `use-cell-value-sync.ts` 中 dispatch 不 await，错误被静默吞没                                                                                  |
