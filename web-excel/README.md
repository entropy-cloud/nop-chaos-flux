# Web Excel

一个独立的纯前端表格应用，直接放在当前目录下，不依赖当前仓库的现有代码。

## 功能

- 区域选择，支持鼠标拖选、整行选择、整列选择、全表选择
- 类 Excel 的底部工作表标签
- 多工作表，支持新增、切换、双击重命名、删除当前工作表
- 公式输入栏，可编辑当前单元格内容
- 基础公式计算，支持 `=A1+B1*2`
- 区间函数：`SUM`、`AVG`、`MIN`、`MAX`、`COUNT`，例如 `=SUM(A1:A10)`
- 复制、剪切、粘贴，支持将单值填充到整片选区
- 样式工具栏：字号、粗体、斜体、下划线、对齐、字色、填充色、清除样式
- 列宽和行高拖拽调整
- 表格行列数量调整
- 浏览器本地保存
- 导入/导出 JSON（兼容旧版单工作表数据）
- 导出当前工作表 CSV

## 使用

直接用浏览器打开 `index.html` 即可。

如果想通过本地静态服务打开，可以在该目录运行任意静态文件服务，例如：

```bash
npx serve .
```
