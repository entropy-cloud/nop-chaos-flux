# Manual Test Issues — 2026-05-18

## 1. flux basic 表格前几行 Inspect 按钮无法稳定弹出对话框

- **所属模块**: playground / flux basic / table row action / scoped slot region / surface runtime
- **问题描述**: 在 `http://localhost:5173/#/flux-basic` 页面最后一个表格里，最后一行的 `Inspect` 按钮可以弹出 `User Details` 对话框，但 `admin` 和 `editor` 所在行的 `Inspect` 按钮无法稳定弹出，表现为点击后无对话框或行数据异常。
- **预期行为**: 任意一行的 `Inspect` 按钮都应稳定打开对话框，并展示当前行的用户名和邮箱。
- **发现方式**: 人工测试
- **状态**: 已修复（最终根因是 table `buttons` / `cell` 等参数化 region 渲染时未传入当前 `rowScope`，多行复用了列级 fragment scope，导致 `$slot.record` 被后续行覆盖或清空；`packages/flux-renderers-data/src/table-renderer/table-body-row-rendering.tsx` 已让这些 region 以 `rowScope` 渲染，`apps/playground/src/pages/fluxBasicPageSchema.json` 同时通过 `openDialog.args.data` 把当前行数据快照进 dialog scope，再由 dialog body 读取 `${username}` / `${email}`；已增加组件级、页面级和 Playwright 回归测试）
