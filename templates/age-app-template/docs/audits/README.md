# Audits Index

Use this directory for audit methodology and audit records.

Start with `00-audit-execution-guide.md`.

Plan and closure audits are mandatory for created plans unless the plan explicitly qualifies for the micro-plan exception. Store audit records here when they are non-trivial, disputed, or useful for future sessions; small audit notes may be recorded in the plan or daily log.
